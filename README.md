# 🧪 ChemData Pro - Complete Chemistry Toolkit

**ChemData Pro** is a comprehensive, interactive web-based chemistry toolkit designed for students, educators, and researchers. It combines a complete periodic table of 118 elements, powerful chemical calculators, data analysis tools, and real-time PubChem API integration—all in one responsive platform.

## ✨ Features

### 🔬 Periodic Table
- Complete 118 elements with accurate properties
- Color-coded categories (Alkali Metals, Halogens, Noble Gases, etc.)
- Click any element to view: atomic radius, electronegativity, ionization energy, melting/boiling points, density, oxidation states, and electron configuration
- Lanthanides and Actinides displayed separately

### 🧮 Chemistry Calculators (with step-by-step workings)
- Molar Mass Calculator
- pH Calculator
- Percent Composition
- Dilution Calculator (C₁V₁ = C₂V₂)
- Temperature Conversion (Celsius, Fahrenheit, Kelvin)
- Oxidation State Finder

### ⚖️ Gas Laws
- Boyle's Law (P₁V₁ = P₂V₂)
- Charles's Law (V₁/T₁ = V₂/T₂)
- Gay-Lussac's Law (P₁/T₁ = P₂/T₂)
- Combined Gas Law (P₁V₁/T₁ = P₂V₂/T₂)
- Ideal Gas Law (PV = nRT)

### 🔬 Advanced Chemistry Tools
- Stoichiometry Calculator
- Reaction Balancer
- Quantum Number Calculator
- Radioactive Decay Calculator

### 💊 Compound Search (PubChem API)
- Search any chemical compound by name
- View molecular structure (2D image)
- Pharmacological properties (therapeutic class, mechanism of action)
- Biological activities (targets, enzymes, cellular effects)
- Toxicity and safety information
- Drug-likeness assessment (Lipinski's Rule of Five)
- Therapeutic uses and dosage information

### 📊 Data Analysis Suite
- Descriptive Statistics (mean, median, mode, variance, standard deviation)
- Linear Regression with R² calculation
- Pearson Correlation Coefficient
- Interactive Charts (Bar, Line, Pie, Scatter)
- Customizable chart titles and axis labels

### 🔐 User Authentication
- Sign up / Login system
- LocalStorage-based user management
- Welcome banner with username display
- Logout functionality

### 🎨 Additional Features
- Dark Mode Toggle
- Printable Results
- Responsive Design (Mobile & Desktop)
- Smooth Animations
- Professional Terms & Conditions and Privacy Policy pages

## 🛠️ Technologies Used
- HTML5
- CSS3 (with Flexbox & Grid)
- JavaScript (ES6+)
- Chart.js for data visualization
- Font Awesome for icons
- Google Fonts (Inter)
- PubChem API for compound data
